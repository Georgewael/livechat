> **(**)** member يقدر يبعت خاص لـ VIP وفوق بس (مش لـ member أو guest)

## ترتيب الرتب (من الأعلى للأدنى)
1. 👑 **Owner** — المالك، صلاحيات كاملة
2. ⭐ **Moderator** — مشرف، يكتم/يطرد/يحذف رسائل الرتب الأدنى
3. 🎖️ **Host** — هوست، يكتم member وguest فقط، ينشئ غرف
4. ✨ **VIP** — عضو مميز، ينشئ غرف، يرسل خاص لأي حد
5. **Member** — عضو عادي، يرسل خاص لـ VIP وفوق فقط
6. **Guest** — زائر، يكتب في العام فقط، بدون خاص أو غرف

## الملفات المضافة/المعدلة
- `permissions.js` — ملف الصلاحيات المركزي (جديد)
- `server.js` — تطبيق فحوصات الصلاحيات على كل الأحداث
- `public/client.js` — إخفاء/إظهار أزرار الـ UI حسب الصلاحية + رسائل الخطأ
- `public/index.html` — إضافة id لقسم إنشاء الغرفة
