# 💬 شات لايف v2.0

## تشغيل المشروع

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل السيرفر
npm start

# 3. افتح المتصفح
http://localhost:3000
```

> للتطوير: `npm run dev` (يحتاج nodemon)

## بيانات المالك (افتراضية)

## هيكل الملفات
```
livechat/
├── server.js          ← السيرفر + Socket.io + API
├── db.js              ← قاعدة البيانات SQLite
├── auth.js            ← مصادقة المالك
├── package.json
└── public/
    ├── index.html     ← واجهة الشات
    ├── login.html     ← صفحة الدخول
    ├── style.css      ← التصميم
    └── client.js      ← منطق الفرونت
```
